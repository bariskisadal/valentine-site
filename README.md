## Kurulum

```bash
npm install
npm run dev
```

## Email gönderimi (opsiyonel)

EmailJS kullanmak isterseniz `.env.example` dosyasını `.env` olarak kopyalayıp
kendi `SERVICE_ID`, `TEMPLATE_ID` ve `PUBLIC_KEY` değerlerinizi ekleyin.

EmailJS ayarlı değilse uygulama `mailto:` ile cihazın mail uygulamasını açar.
